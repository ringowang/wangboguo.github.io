Atom实用快捷键

By wangboguo

2016-10-25 Updated:2016-11-03

Contents
  1. shift + ⌘ + p 输入命令
  2. ⌘ + p 找文件
  3. ctrl + g 到某一行
  4. ⌘ + r 搜某个函数（方法）
  5. shift + ⌘ + { /（或者） } 到前/后一个文件
  6. ⌘ +]/[ 整行缩进或回退
  7. ⌘ + enter 新插入一行(即便光标不在尾端)
  8. ⌘ + shift + enter 在本行上面插入一行
  9. ctrl + shift + k 删除本行，光标行数不变
  10. ctrl + ⌘ + up/down 把一行向上/下移动
  11. shift + ⌘ + d 在下面复制一行新的
  12. ⌘ +j 把下一行移到本行最后
  13. ⌘ +f 搜索
  14. ⌘ +g 在搜索后，跳到下一个
  15. shift + ⌘ + f 在本工程里搜索
  16. ⌘ + e 光标在哪个word上就搜索和它一样的
  17. ctrl + ⌘ + [ / ] 现在还没用，没有fold block
  18. option + shift + ⌘ + [ / ] 折叠/打开这一整段
  19. ⌘ + 1-9 进入第1-9个文档
  20. ⌘ +d 选中光标指的词
  21. ⌘ +l 选中这一行
  22. option + shift + left/right 选中光标左边/右边到词的开头/结尾
  23. shift + ⌘ + left/right 选中光标左边/右边到这句的开头/结尾
  24. ⌘ +\ 折叠边栏
  25. ctrl + ⌘ + f 全屏
  26. ⌘ + k, left/right/up/down 开一个新的区域在上下左右
  27. ctrl + t 光标两头调换位置
  28. option + delete 删到光标所指词的开头
  29. shift + ⌘ + up/down 选中光标上/下的全部
  30. optinon + command + s 保存所有改动

在 Atom 里，⌘k 默认被设置成了一些双组合键的第一个，比如，你按过一次 ⌘k 之后再按右方向键 → 的话，你当前的输入区域就会在一个新生成的右侧面板（Right Pance）上打开一份
